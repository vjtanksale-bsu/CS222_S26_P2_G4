# CS222_S26_P2_G4
User Story 3: Select Courses

As a student, I want to enter my selected course numbers so that the system can
create my semester schedule.

Acceptance Criteria:
1.The system allows the student to enter course numbers after specifying the
number of courses they wish to register for.

2.The student can enter exactly the number of course numbers requested.

3.The system verifies that each entered course number exists in the list of offered
courses.

4.The system requires all selected course numbers to be distinct.

5.The system displays an error message if a duplicate course number is entered.

6.The system displays an error message if an invalid course number is entered.

7.After all valid course numbers have been entered, the system accepts the course
selections and proceeds to the next step.
